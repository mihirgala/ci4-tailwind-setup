Project folder structure
layout shall be followed in
views/layouts/layout.php
views/pages/ all pages will be here everything in the page will be broken down into sections
views/sections/ for e.g hero.php section2.php section3.php etc

css file shall be in 
assets/css/input.css only if to be used throughout the website other wise it shall be used by <style></style> tag inside the view file itself

javascript shall be in
assets/js/script.js only if to be used throughout the website other wise it shall be used by <script></script> tag inside the view file itself

Reusable sections shall be made using array loops/maps.
